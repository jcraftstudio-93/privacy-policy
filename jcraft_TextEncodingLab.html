1) Overview / 개요

Text Encoding Lab is a utility app that converts text using features such as Base64, URL encoding, HTML entities, ROT13, Caesar/Atbash ciphers, and Morse code.

Text Encoding Lab은 Base64, URL 인코딩, HTML 엔티티, ROT13, 시저/앳바시, 모스부호 등의 기능으로 텍스트를 변환하는 유틸리티 앱입니다.
