개인정보처리방침

본 개인정보처리방침은 PDF Splitter 앱(이하 “앱”)에 적용됩니다.

중요 안내: 본 앱은 사용자의 편의를 위한 PDF 분할 및 파일 처리 기능을 제공합니다. 사용자는 중요한 문서에 대해 별도의 백업을 유지해야 하며, 개발자는 사용자의 기기 상태, 파일 손상, 앱 삭제, 운영체제 제한, 저장소 오류, 사용자 조작 실수, 제3자 앱 동작 등으로 인해 발생할 수 있는 데이터 손실이나 간접적 손해에 대하여 법이 허용하는 최대 범위 내에서 책임을 제한합니다.

1. 운영자 정보

개발자 정보는 Google Play 스토어의 개발자 정보 항목을 참고해 주세요.

2. 수집하는 정보

본 앱은 기본적으로 사용자가 선택한 PDF 파일을 분할하는 기능을 제공합니다. 앱 운영 과정에서 아래와 같은 정보가 처리될 수 있습니다.

  • 사용자가 직접 선택한 파일 정보(파일명, 경로, 크기 등)
  • 앱 내부에서 생성된 분할 결과 파일 정보
  • 광고 제공 및 앱 안정성 확보를 위한 기기 관련 비식별 정보
  • 오류 분석 또는 성능 개선에 필요한 제한적 기술 정보

3. 개인정보의 수집 방식

  • 사용자가 직접 파일을 선택하거나 앱 기능을 사용하는 과정
  • 광고 SDK 또는 운영체제가 자동으로 처리하는 기술 정보
  • 파일 열기, 공유, 저장 등의 기능 사용 과정

4. 정보 이용 목적

  • PDF 파일 분할 및 결과 파일 생성
  • 선택한 파일의 처리 및 결과 제공
  • 앱 성능 유지, 오류 확인 및 안정성 개선
  • 광고 제공 및 서비스 운영

5. 파일 및 저장소 관련 안내

본 앱은 사용자가 선택한 파일을 처리하고, 분할 결과 파일을 기기 저장소 또는 앱이 접근 가능한 위치에 저장할 수 있습니다.

  • 앱 삭제, 저장소 정리, 기기 초기화, 권한 변경, 운영체제 정책 변경 등에 따라 저장된 파일이 삭제되거나 접근이 불가능해질 수 있습니다.
  • 중요한 문서 및 결과 파일은 사용자가 별도로 백업해야 합니다.
  • 개발자는 사용자가 저장하지 않았거나, 백업하지 않은 파일의 보존을 보장하지 않습니다.
  • 파일 공유 또는 외부 앱으로 열기 기능 이용 시, 해당 외부 서비스 또는 외부 앱의 처리 방식은 각 서비스 제공자의 정책을 따릅니다.

6. 광고 및 제3자 서비스

본 앱은 광고 제공을 위해 제3자 서비스(예: 광고 SDK)를 사용할 수 있습니다. 이러한 제3자 서비스는 광고 제공, 성과 측정, 부정 사용 방지 등을 위해 기기 식별자 또는 비식별 기술 정보를 처리할 수 있습니다.

제3자 서비스의 정보 처리에 대해서는 각 서비스 제공자의 정책이 적용될 수 있습니다.

7. 정보 보관 및 삭제

  • 앱 내부에서 생성되거나 저장된 일부 데이터는 사용자의 기기 내에 보관될 수 있습니다.
  • 사용자는 앱 삭제, 앱 데이터 삭제, 파일 삭제 등을 통해 일부 정보를 직접 제거할 수 있습니다.
  • 단, 외부 저장소 또는 외부 앱으로 복사·공유된 파일은 사용자가 직접 관리해야 합니다.

8. 아동의 개인정보

본 앱은 아동을 주요 대상으로 설계되지 않았습니다. 보호자는 아동이 앱을 사용하는 경우 파일 처리 및 공유 기능 사용에 주의해 주시기 바랍니다.

9. 보안에 관한 안내

개발자는 합리적인 범위에서 앱의 안정성과 보안을 고려하지만, 인터넷 환경, 기기 보안 상태, 운영체제 제한, 루팅 또는 탈옥 환경, 악성코드, 외부 앱 충돌 등 모든 상황에 대해 완전한 보안을 보장할 수는 없습니다.

  • 중요 문서는 별도로 백업해 주세요.
  • 신뢰할 수 없는 앱과의 공유는 주의해 주세요.
  • 사용자의 기기 보안 상태에 따라 파일 보호 수준은 달라질 수 있습니다.

10. 책임의 제한

본 앱은 “있는 그대로(as is)” 제공됩니다. 개발자는 관련 법령이 허용하는 최대 범위 내에서 다음 사항에 대한 책임을 제한하거나 부담하지 않을 수 있습니다.

  • 파일 손상, 누락, 중복 저장, 저장 실패, 분할 실패
  • 앱 삭제, 기기 변경, 운영체제 업데이트, 저장소 오류로 인한 데이터 손실
  • 외부 앱, 외부 저장소, 공유 기능 사용 과정에서 발생한 문제
  • 간접 손해, 특별 손해, 부수적 손해, 결과적 손해, 영업 손실, 기회 손실
  • 사용자의 부주의, 오사용, 잘못된 설정, 백업 미실시로 인한 문제

사용자는 본 앱을 사용하기 전에 중요한 문서를 직접 백업하고, 결과 파일을 스스로 확인할 책임이 있습니다.

11. 정책 변경

본 개인정보처리방침은 필요에 따라 변경될 수 있습니다. 중요한 변경이 있는 경우 업데이트된 정책이 본 페이지에 게시될 수 있습니다.
